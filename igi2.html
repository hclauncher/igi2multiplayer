System Requirements

This game is a first-person tactical shooter PC game developed by Innerloop Studios and published by Codemasters in 2003. It features both Single Player and Multiplayer modes.

Minimum Requirements

  • CPU: Pentium III or Athlon 700MHz Processor
  • RAM: 128MB RAM
  • GPU: 32MB 3D Accelerated Video Card
  • DirectX: Version 8.1
  • OS: Windows 98 / ME / 2000 / XP / 7 / 10
  • Storage: 1.9GB Hard Disk Space
  • Network: 56Kbps Modem (Modem Play)

Recommended Requirements

  • CPU: Pentium 4 or Athlon 1.2GHz Processor
  • RAM: 512MB RAM
  • GPU: 64MB 3D Accelerated Video Card
  • DirectX: Version 8.1
  • OS: Windows 98 / ME / 2000 / XP
  • Storage: 1.9GB Hard Disk Space
  • Network: 512Kbps Modem (for Hosting)
