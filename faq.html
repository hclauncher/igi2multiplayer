Frequently Asked Questions

How to download IGI-2 Covert Strike (v1.2)? −
Visit the Homecoming website https://igi2multiplayer.pages.dev, then go to the IGI-2 menu and scroll down to the Click Here or Download button to get the fresh and compressed version of IGI-2 game v1.2.
How to install igi2 covert strike on your machine? +
First download the fresh IGI2 game v1.2, then go to your Download folder. Use any extractor like 7zip or WinRAR to extract the compressed IGI2 file. After extract, run the file — while running, in the popup CMD window it will ask you to Press any key to continue installation. Wait for a few minutes for installation to complete.
How to play IGI-2 Multiplayer? +
Homecoming launched a new project on the Homecoming website called HcLauncheR. With that, you can easily play IGI2 multiplayer — just download, install, and play!
What is HcLauncheR? +
HcLauncheR is a Homecoming Project developed by SNIPER. It helps players play IGI-2 Multiplayer easily. HcLauncheR is a lightweight software that’s easy to download and install, and works on all kinds of Windows OS — even low-end PCs. It includes features like login system, name change requests, 24/7 contact support, server hosting, and IGI2 map downloader — all in one click.
How to download and install HcLauncheR? +

Download:

Click this link to download HcLauncheR https://igi2multiplayer.pages.dev/hclauncher/ or https://igi2multiplayer.pages.dev

Install:

After download, extract HcLauncheR.zip and run HcLauncheR.msi Windows installer file. Click “Next” and browse to your installation folder (recommended: IGI2-Covert Strike\pc). Choose “Just me” (recommended), then click “Next” again to start installation. Click “Yes” when prompted for admin privilege. Once installed successfully, click “Close” to exit — that’s it!

How to update HcLauncheR? +
HcLauncheR will automatically notify you when an update is available — no need to check manually. When the popup shows, click “OK” to update. If you click “Cancel”, the update will be skipped. Follow the same installation steps as before.
How to change your multiplayer name in HcLauncheR? +
You can request a name change in HcLauncheR after using it actively for one month. To do this: open HcLauncheR → go to the Help menu → choose Request → Name Change form → fill all details correctly to avoid failure.
What is Homecoming? +
Homecoming is the official IGI2 multiplayer gaming community for entertainment purposes. It provides multiple servers from various regions for the best player experience. It includes two major projects — HcLauncheR (Homecoming Launcher) for players, and HcCommandeR (Server Commander) for server control. Homecoming uses master servers to host all IGI2 servers securely, protecting them from DDoS attacks, packet sniffing, and unauthorized packets.
What is HcBan? +
HcBan is a Homecoming Project developed by SNIPER. It helps IGI-2 server owners protect their servers from various types of crashers such as DDOS attacks, packet flooding, no-signature packets, huge or malformed packets, and map crashing exploits.

HcBan is an automated banning system designed with high security to detect and block malicious users in real time. It helps server owners maintain smooth gameplay by automatically banning suspicious players without manual intervention.

IGI-2 Multiplayer and Homecoming Projects

Homecoming Gaming is a passionate community built for fans of IGI-2: Covert Strike — the legendary tactical shooter from Innerloop Studios. Our goal is to revive and modernize the IGI-2 multiplayer experience with secure servers, automatic updates, and dedicated support for players around the world. Through projects like HcLauncheR, HcCommanDeR, and HcBan, we provide an easy and safe way for players to join, host, and protect IGI-2 servers with minimal setup.

The HcLauncheR software allows players to download, update, and launch IGI2 multiplayer effortlessly. It automatically lists active servers, handles name management, and ensures smooth connectivity. For server administrators, HcCommanDeR offers full control over IGI-2 server operations such as kicking, banning, messaging, and monitoring player activity. Both tools are lightweight, compatible with all Windows versions, and designed to keep the classic IGI-2 experience alive on modern PCs.

We also take player safety seriously. The HcBan system continuously monitors incoming packets, detecting crashers, DDoS floods, malformed packets, and map exploiters in real time. With automatic banning and live protection, it ensures that your game sessions remain stable, fair, and enjoyable for everyone.

Whether you’re a returning player or completely new to the IGI universe, Homecoming Gaming offers everything you need — from installation guides and IGI2 map downloads to live community support through Discord and Facebook. Join us today to relive the thrill of IGI-2 Covert Strike multiplayer and be part of a growing worldwide community that keeps this classic game alive for future generations.
